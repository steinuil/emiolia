Emiolia is supposed to be a music sheet manager and reader for the GNOME desktop.

It should support these things, ideally:

- Manage pieces within a library
  - Import PDFs, images, maybe MusicXML
  - CRUD for each piece in the library
  - User metadata: the user should be able to add their own fields
  - Default metadata
    - Title
    - Composer
    - Arranger
    - Lyricist
    - Instruments/voices
    - Year
  - Different versions of a piece
    - e.g. parts that include different instruments, other editions
  - Different types of collections
    - Tags (unordered)
    - Suites (ordered)
    - Setlists (ordered)
  - Filter and search by all fields and metadata
- Multiple libraries
- Slice and dice pieces
  - Crop pages to content
  - Rearrange, add, and delete pages
  - Split a piece into multiple pieces
- Reading pieces
  - Half-page turns
  - Repeats by creating links displayed on pages
  - Set page numbers 
  - Go to next piece in suite/setlist
- Annotating PDFs by drawing, text, symbols
  - This implies adding support for tablets!
- Bookmarks for PDFs
- Integrate with bluetooth pedals and other things
- Syncing the library with some external service
